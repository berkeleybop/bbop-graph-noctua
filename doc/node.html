subgraph([subgraph]) → (graph or null)

Get/set the "contained" subgraph. This subgraph is still considered to be part of the graph, but is "hidden" under this node for most use cases except serialization.

To put it another way, unless you specifically load this with a specialized loader, it will remain unpopulated. During serialization, it should be recursively walked and dumped.
